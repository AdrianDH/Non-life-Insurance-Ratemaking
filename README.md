# 产险精算定价

保险是经营风险的行业，风险的评估和定价是保险公司为核心的竞争力。本书以保险业为研究对象，讨论了相应的风险模型及其应用，主要包括损失概率、损失次数、损失金额和累积损失的分布模型以及它们的预测模型，同时还探讨了巨灾损失和相依风险的建模问题。在实证研究中，以 R 语言为计算工具，提供了详细的程序代码，方便读者再现完整的计算过程。

本书适合风险管理、保险与精算等相关专业的高年级学生、研究人员或从业人员参考。
